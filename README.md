# Notas:
Detalles de aplicación básica de Node:
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base de la tabla de multiplicar            [número] [requerido]
  -l, --listar   Imprime la tabla en consola         [booleano] [defecto: false]
  -h, --hasta    Límite de la tabla de multiplicar          [número] [requerido]
```
